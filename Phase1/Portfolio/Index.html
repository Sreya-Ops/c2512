Projects

OPTIMISED DESIGN OF SMART CHARGING STATION FOR EV

Objective of the project is to design a Solar powered Smart charging station for Electric Vehicle with Multiple port and Dynamic pricing facility.This project presents an in-depth exploration of the optimized design of Smart Charging Station for EVs, focusing on enhancing user experience,energy efficiency, and grid integration.

IoT INTEGRATED VEHICLE CONTROL SYSTEM USING CAN PROTOCOL

The project aims to develop an IoT-based Smart Vehicle Control System using NodeMCU ESP32,STM32F407, CAN Controller, and sensors for real-time monitoring of RPM and fuel levels, along with remote vehicle control via Wi-Fi and the Blynk app.

OBSTACLE DETECTING ROBOT USING PIC MICROCONTROLLER

We have developed an Obstacle-detecting robot based on the PIC16F887 microcontroller using Ultrasonic sensors to detect obstacles in its path and takes appropriate actions to avoid them. The microcontroller processes input from the sensors,and controls the robot's motors to navigate around obstacles. This system is useful in applications like autonomous navigation and robotics education.

RFID AND KEYPAD BASED DOOR LOCK SYSTEM USING STM

The objective of our project is to create a secure and efficient access control mechanism that leverages both RFID technology and keypad input for user authentication,using STM32F407 microcontroller.
